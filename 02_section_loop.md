# Module 2: Section Loop

## Purpose
Summarize each major section of the research paper.

## Responsibilities
- Iterate through all required sections:
  Abstract, Introduction, Methodology, Results, Discussion, Conclusion
- Summarize each section with:
  - Maximum 150 words
  - Audience-specific phrasing rules
- Extract key points only from original text (no hallucinations)
- Store structural and glossarial information for later modules

## Inputs
- Cleaned sections from Module 1
- Target audience

## Outputs
- Summaries for each section
- Glossary candidates
- Alerts for empty or very short sections
